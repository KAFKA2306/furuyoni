    ユキヒ

    • 権能: 傘 (Umbrella)
    • 難易度: ★★★★☆
    • 得意[間合](../mechanics.md#range): 0-2 (閉) / 4-6 (開)

    その傘は、盾となり、矛となる

    相手のメガミやデッキタイプに合わせて、自分が「アグロ（攻撃重視）」になるか「コントロール（防御重視）」になるかを選択できる点が最大の強みです。

    物語の起源：ユキヒ

    「開くも閉じるも、私の自由。この傘が紡ぐのは、あなたへの引導よ」

    ユキヒは、変幻自在の「傘」を操り、二つの顔を持つメガミです。 彼女の起源は、移ろいゆく季節の境界や、物事の「表」と「裏」が交差する瞬間にあります。可憐な少女の如き微笑みの裏に、冷徹な暗殺者の鋭利な牙を隠し持っています。 物語では、その二面性ゆえに捉えどころのない、しかし無視できない存在感を放つトリックスターとして描かれます。彼女の振るう傘は、ある時は降り注ぐ災厄を遮る絶対的な盾となり、またある時は相手の急所を一突きにする必殺の矛となります。 「開閉」という力は、彼女が状況に応じて己の在り方（距離感）を峻烈に変え、相手の計算を根底から裏切り続ける「変化」の美学を象徴しています。

    キーワード能力: 傘 (Open/Close)

    ユキヒは常に「傘が開いている(開)」か「傘が閉じている(閉)」のどちらかの状態にあります。 決闘開始時にどちらの状態から始めるかを選択できます。

    選択のタイミング

    傘の状態（開/閉）は、双掌繚乱（メガミの公開）後の「1ターン目の開始フェイズ（初期手札の引き直し前）」に決定します。相手のメガミを見てから、どちらが有利かを冷静に判断しましょう。

    • 閉 (Closed) - 暗殺者:
      • 得意間合: 0-2
      • 役割: 近距離バースト、潜り込み
      • 特徴: 相手の懐に入り込み、『ゆらりび』などの必殺の一撃を狙う攻撃的なスタイル。一時的なリソース（集中力や手札）を吐き出して大ダメージを狙います。
    • 開 (Open) - 舞踏家:
      • 得意間合: 4-6
      • 役割: 中距離コントロール、レンジロック
      • 特徴: 相手を寄せ付けず、一方的に攻撃を加える防御的なスタイル。毎ターンの安定した行動と、相手の攻撃リソースを枯らす持久戦を得意とします。

    状態の切り替え: 特定のカードの使用や効果によって、傘の状態は反転します。この「スイッチ」のタイミングを制御することが、ユキヒ使いの腕の見せ所です。 安易な開閉は、相手に隙を晒すことになります。「次の相手のターン、自分はどういう状態でいたいか？」を常に意識しましょう。

    シーズン10改訂情報のポイント

    シーズン10（および9-2）のカード更新により、ユキヒは以下の強化を受けています。

    • S1 はらりゆき: 射程が3-6に拡大（旧3-5）。
      • これにより、相手のレンジロック（間合6など）に対しても、閉状態から攻撃を当ててライフダメージを稼ぎやすくなりました。
    • 集中力の重要性: カード更新により、一部の行動で「集中力」を得やすくなるなどの調整が入っています（※具体的なカードテキストは最新のものを参照してください）。

    これらの変更により、ユキヒは「中距離での撃ち合い」において以前よりも強力な立ち位置を確立しています。

    通常札解説

    ユキヒのカードの多くは、現在の傘の状態によって名前と効果が変化します。

    N1 / ふくみばり

    役割: 攻撃 / 反撃

    • 閉：
      • 間合2で撃てる3/1攻撃。
      • ユキヒの貴重な中打点です。「」等で接近した直後に放つ基本技であり、ライフへ圧力をかける重要リソースです。
    • 開：ふくみばり
      • 相手の攻撃に対応して[2/2]を返す強力なカウンター。
      • 相手はこれを警戒して「2/2を食らっても良いタイミング」でしか攻撃できなくなります。存在自体が防御壁です。


    N2 / ねこだまし

    役割: バースト / 妨害

    • 閉：
      • 条件付き[4/3]などの高打点化。
      • 非常に高い瞬間火力を持ちますが、自傷ダメージのリスクがあります。リーサル（とどめ）を見据えた局面での切り札として運用します。
    • 開：ねこだまし
      • ダメージを与えず、相手のオーラや集中力を奪う妨害札。
      • 相手の「攻撃する前の準備」を崩すのに最適です。


    N3 ふりはらい /

    役割: 拒否 / 吸引 ※最重要カード

    • 閉： (Pull)
      • 「間合を2つ近づける（相手のオーラを無視して）」という強烈な移動効果。
      • 遠距離メガミ（ヒミカなど）を無理やりキルレンジに引きずり込むための生命線です。
    • 開：ふりはらい (Push)
      • 「間合を2つ離す（相手のオーラを無視して）」。
      • 近距離メガミ（ユリナなど）の攻撃可能範囲から脱出し、完封するための絶対的な拒否手段です。


    N4 ふりまわし /

    役割: 範囲攻撃 / 一点突破

    • 閉：
      • 至近距離での一点突破。相手の「対応」を許しにくい鋭い攻撃です。
    • 開：ふりまわし
      • 間合3-5全体をカバーする範囲攻撃。
      • クリンチ（密着）された状態から、とりあえず振って間合を調整する際にも重宝します。


    N5

    役割: スイッチ / 潤滑油

    • 効果: 傘の状態を変更する。
    • 運用: ユキヒのエンジンの要。攻撃行動の終わり（閉状態）にこれを使って「開」にしてターンを終了し、相手ターン中の防御力を高める動きが基本かつ強力です。
    • このカードを毎ターン使えるように回すことが、ユキヒの安定運用への近道です。


    N6 ひきあし /

    役割: ステップ / ダイブ

    • 閉：
      • 相手の懐へ飛び込む移動技。オーラダメージを軽減しながら前進できます。
    • 開：ひきあし
      • 安全圏への離脱。
      • 間合調整のためだけでなく、相手の攻撃を回避するための「事前準備」としても機能します。


    N7

    役割: 接続 / 妨害

    • 解説: 特殊なバフ/デバフカード。
      • 相方のメガミとのシナジーを形成するための「接着剤」のようなカード。特定のコンボデッキで真価を発揮する玄人向けの一枚。


    切札解説

    S1 はらりゆき

    コスト: 1 (消費フレア) 種別: 攻撃 / 対応

    解説: 攻防一体の優秀な切札。 シーズン10より射程が3-6に強化され、より広い範囲で運用可能になりました。 消費フレア1という軽さで、相手のステップ対応や攻撃に合わせて撃つことができます。 「あと1点が足りない」という場面や、「相手の攻撃を止めたい」場面のどちらでも使えるため、腐ることがありません。 特に間合6でのレンジロック合戦において、このカードの存在が大きな圧力となります。


    S2 ゆらりび

    コスト: 5 種別: 攻撃 (超必殺技)

    解説: ユキヒ最強の矛。 「閉状態」かつ「間合0-2」という厳しい条件はありますが、その破壊力はゲームエンド級。 「対応不可」を付与するサポート（他メガミのカード）と組み合わせることで、確定リーサルを作り出すことができます。 このカードを見せるだけで、相手はライフ受けができなくなり、プレイングを大きく縛ることができます。

    「ゆらりび」対策への対策

    強力無比な攻撃ですが、間合が「0-2」に限られているため、相手が『』などの対応カードを持っていると簡単に適正距離外へ逃げられてしまいます。 相手の「対応」を吐かせきった後に振るか、オボロの『』などで手札を破壊し、逃げ道を塞いでから叩き込む慎重さが必要です。


    S3 くるりみ

    コスト: 2 種別: 行動 / 対応

    解説: 防御の要。 「攻撃を無効化（ステップなどで回避）」しつつ、「傘の状態を切り替える」ことができます。 相手のターン中に傘を開閉できる数少ない手段であり、これによって相手の計算を狂わせることができます。


    S4 ひらりおり

    コスト: 3 種別: 行動 / 対応

    解説: 究極の回避技。 間合を大きく動かすことで、相手の攻撃を「適正距離外」にし、不発に終わらせることができます。 特に「月影落」のような「適正距離が狭く、強力な攻撃」に対するメタカードとして機能します。


    A1 特殊カード (Another Edition)

    ユキヒのアナザー版（A1）は、通常版の以下のカードを入れ替えて使用します。 A1は「社交」をテーマにしており、より盤面への干渉力が高まっています。

    N2 (A1) ちからぞえ / おどしつけ

    役割: 妨害 / 支援 * 閉：ちからぞえ * 間合2-7、ダメージ2/1。相手の山札が2枚以上ならダメージの代わりに2枚伏せ札にする。攻撃しながらリソースを削るテクニカルな札です。 * 開：おどしつけ * ダメージではなく、相手に「畏縮」などのプレッシャーを与える妨害札。


    N4 (A1) よこいと / たていと

    役割: 範囲サポート / 移動 * 閉：たていと * 縦の糸（間合）を制御する構築札。 * 開：よこいと * 適正距離の拡大（遠近+1/-1）。相方の攻撃範囲を劇的に広げ、本来当たらない間合からの奇襲を可能にします。使用後は山札の底に戻るため、再利用が容易です。


    2つの主要アーキタイプ

    ユキヒの構築は、どちらの形態を重視するかで大きく2つに分類されます。

    1. クリンチ・ユキヒ (閉状態重視)

    「潜って、刺す。」 積極的に前に出て、間合0-2に張り付くスタイルです。

    • 基本戦術:
      1. 開幕から前進し、間合2へ。
      2. 『』『』で相手の離脱を許さず、粘着します。
      3. 『』と相方の近距離攻撃でライフを削り取ります。
    • フィニッシュ: 相手のオーラが枯渇した瞬間を見逃さず、『ゆらりび』を叩き込みます。あるいは、相方の大技で沈めます。

    オーラ溢れに注意

    積極的に間合2以下に潜り込む際、相手のオーラを剥ぎ取ることが多いですが、自分のオーラが最大の時にさらに纏おうとすると「オーラ溢れ」が発生し、リソースが無駄になります。前進する際は、あらかじめ自分も攻撃を振っておくなど、オーラの「空き」を作っておくのが効率的です。

    • 強み: 遠距離メガミに対する圧倒的な制圧力。
    • 弱み: 近距離での殴り合いに強いメガミ（ライラ等）には、火力負けするリスクがあります。

    2. レンジロック・ユキヒ (開状態重視)

    「舞って、いなす。」 間合4-6を維持し、相手に指一本触れさせないコントロールスタイルです。

    • 基本戦術:
      1. 『ふりはらい』『ひきあし』で徹底的に距離を取ります。
      2. 中距離から『ふりまわし』や相方の射撃攻撃で一方的にダメージを与えます。
      3. 相手がリソースを吐いて接近してきたところに『ふくみばり』のカウンターを合わせます。
    • フィニッシュ: 相手の焦りを誘い、無理な前進に対してリソース差で押し切ります。
    • 強み: 近距離特化メガミ（ユリナ等）を完封しやすい。
    • 弱み: こちらから攻める速度が遅いため、準備の整ったコンボデッキには押し負けることがあります。

    おすすめの組み合わせ (Pairs)

    オボロ (戦術名: みかはらり)

    スタイル: 設置型コンボ

    • 概要: オボロの設置技と、ユキヒのスイッチギミックが噛み合ったテクニカルなペア。
    • 名前の由来: ユキヒの旧名「ミカ」と、切り札『はらりゆき』、そしてオボロの『壬蔓（みかづら）』の「みか」を掛け合わせた、古参プレイヤーにも親しまれる通称です。

    • シナジー:

      • 設置: オボロの『』を設置しておくことで、相手の手札（対応札）を破壊します。

      • 無防備な相手に: 対応札を失った相手に対し、ユキヒの『ゆらりび』やオボロの『熊介』などの高火力を、安全かつ確実に叩き込みます。

      • 壬蔓: オボロの『壬蔓』が、ユキヒの消費フレアをサポートし、手数と柔軟性を底上げします。 → 詳細な攻略（忍傘）

    ユリナ (戦術名: 刀傘ビート)

    スタイル: スタンダード / ビートダウン

    • 概要: 全距離対応の王者。ユリナが「剛」、ユキヒが「柔」を担当します。
    • シナジー:
      • 間合の穴がない: ユリナが得意な3-4、ユキヒが得意な0-2と4-6により、どの間合にいても有効打を持てます。
      • ダブルフィニッシャー: 『月影落』(超火力)と『ゆらりび』(超火力)の両方を警戒させることで、相手の防御リソースを過剰に吐かせることができます。
      • 初心者推奨: お互いの弱点を補い合っており、基本的な立ち回りを学ぶのに最適なペアの一つです。 → 詳細な攻略（刀忍） (※刀傘は刀忍と同様のアグロ戦術)

    トコヨ (戦術名: 扇傘コントロール)

    スタイル: 徹底防御 / レンジロック

    • 概要: 相手の心を折る鉄壁の要塞。
    • 戦術:
      • トコヨの『無窮ノ風』などで相手の手札を奪い、ユキヒの開状態で接近を拒否します。
      • 相手が攻めあぐねている間に、少しずつライフ差を広げて判定勝ちなどを狙います。
      • 時間切れになりやすいため、大会などでは時間管理が重要になる玄人向けの構成です。 → 詳細な攻略（忍傘） (※忍傘に並ぶコントロールの代表)

    ヒミカ (戦術名: 銃傘レンジロック)

    スタイル: 遠距離ビートダウン / コンボ

    • 概要: 遠距離（ヒミカ）と中近距離（ユキヒ）の攻撃範囲を完備した攻撃的ペア。

    • シナジー:

      • クリムゾンゆらりび: ヒミカの『クリムゾンゼロ』（間合0-2で攻撃可能になる行動カード）と、ユキヒの『ゆらりび』（間合0-2超火力）を組み合わせる必殺コンボ。『クリムゾンゼロ』によりユキヒの苦手な前進を補い、一気に密着して致命傷を与えます。

      • 完全なレンジ支配: 開状態ของユキヒ（間合4-6）とヒミカ（間合5-8）により、ほぼ全域で攻撃権を主張できます。相手が近づけば『ゆらりび』、離れれば銃撃が襲います。

      • A1「社交」シナジー: ユキヒA1の『よこいと』でヒミカの銃攻撃の適正距離を広げることで、さらに対応されにくい間合からの射撃が可能になります。
      • 弱点: 『ゆらりび』コンボは強力ですが、対応（ステップなど）には脆いため、相手の対応札の有無を見極める必要があります。 → 詳細な攻略（薙銃） (※薙銃と同様の遠距離支配)

    サイネ (戦術名: 薙傘テンポ)

    スタイル: ミッドレンジ / テンポ

    • 概要: サイネのリーチとユキヒの柔軟性を組み合わせた、対応力重視のテンポデッキです。
    • シナジー:
      • 「閉」のカバー: サイネが苦手とする「攻撃後の隙（リソース不足）」を、ユキヒの「閉状態」での『はらりゆき』や『しこみばり』でカバーします。
      • ソフト・レンジロック: お互いに『後退』や『ひきあし』のような移動札が豊富なため、間合5-7を維持し続ける「ソフトなレンジロック」が得意です。
      • 八相の維持: ユキヒのカードで間合を調整し、サイネが安全に八相（オーラ1以下）で暴れる盤面を作ります。
      • 弱点: 決定打に欠ける場合があり、ジリ貧になる前に『律動弧戟』や『ゆらりび』を通す計画性が必要です。

    チカゲ（千紫万紅ビート / 傘毒）

    チカゲの『泥濘』で相手を足止めし、ユキヒが理想的な間合から『ゆらりび』や『つきさし』を叩き込みます。 S10において、チカゲが間合1で『毒針』を刺せるようになったため、ユキヒの「閉じた傘（近距離）」での殺傷力が劇的に向上しました。

    戦術の核心

    「社交」よこいとによる遠距離支配

    A1ユキヒを採用する場合、開状態の『よこいと』が相方の性能を120%引き出します。特にヒミカなどの遠距離メガミと組む際、本来の射程外（間合2〜4の隙間など）からの弾幕が可能になり、レンジロックの強度が飛躍的に高まります。

    リソース管理の取捨選択

    ユキヒは間合操作に長けますが、手札と集中力の消費が激しいメガミです。「この攻撃は受けてフレアにし、次のターンのバーストに繋げる」といった、長期的なダメージコントロールを意識しましょう。 単なる守備的な立ち回りだけでは、シーズン10の火力環境を生き抜くことは困難です。
