# 桜花結晶の移動ルール

このゲームの全リソース（距離・盾・体力・エネルギー）は**桜花結晶**という共通の単位で表される。
結晶は消滅せず循環する。この理解がゲームの全戦術判断の土台になる。

---

## 1. 5つのエリア

| エリア | 説明 | 上限 | 初期値 |
|:---|:---|:---:|:---:|
| **間合** | 二人の間の距離 | なし（実質的に制限あり） | 10 |
| **オーラ** | プレイヤーの盾 | **5** | 3 |
| **ライフ** | プレイヤーの体力 | - | **10** |
| **フレア** | 切札のコスト | なし | 0 |
| **ダスト** | 共有の結晶プール | なし | 0 |

### 追加のエリア（特定のメカニクス）
- **付与札上の結晶**: 付与札を使用した際に配置される。開始フェイズに1つずつダストへ。
- **造花結晶**（サリヤ専用）: サリヤ固有の結晶。燃焼して消費する。

---

## 2. 基本動作（4種）

基本動作を行うには**コスト**が必要：**集中力1** or **手札1枚を伏せる**

| 動作 | 移動方向 | 制約 |
|:---|:---|:---|
| **前進** | 間合 → 自オーラ | 間合が0なら不可。自オーラが5なら不可。 |
| **後退** | 自オーラ → 間合 | 自オーラが0なら不可。 |
| **纏い** | ダスト → 自オーラ | ダストが0なら不可。自オーラが5なら不可。 |
| **宿し** | 自オーラ → 自フレア | 自オーラが0なら不可。 |

### 重要な注意
- 前進は「相手に近づく」行為。間合が小さいほど近い。
- 後退は「相手から離れる」行為。
- 基本動作は1回につき結晶**1つ**を移動する。
- 基本動作はメインフェイズの標準行動中に行う（全力行動中は不可）。

---

## 3. ダメージによる結晶移動

攻撃がヒットした場合のダメージ処理：

### X/Y 表記
- **X** = オーラダメージ / **Y** = ライフダメージ
- **受け手（被攻撃者）がどちらで受けるか選択する**。

### オーラ受け
- 自オーラからダストへ **X個** 移動。
- **オーラが不足する場合**: 
  - オーラが X 未満なら**ライフ受けを強制**（オーラ受けは選択不可）。
  - つまりオーラが0の状態ではライフでしか受けられない。

### ライフ受け
- 自ライフからフレアへ **Y個** 移動（※ダストではなくフレアに行く！）。

### 特殊ケース
- **-/1** のようなダメージ：オーラ側がハイフンならライフ受け強制。
- **ダメージが0の場合**: ダメージとしては解決するが結晶は移動しない。

---

## 4. 結晶移動の全パターンまとめ

| 事象 | 移動元 → 移動先 |
|:---|:---|
| 前進 | 間合 → 自オーラ |
| 後退 | 自オーラ → 間合 |
| 纏い | ダスト → 自オーラ |
| 宿し | 自オーラ → 自フレア |
| ダメージ(オーラ受け) | 被攻撃者オーラ → ダスト |
| ダメージ(ライフ受け) | 被攻撃者ライフ → **被攻撃者フレア** |
| 再構成コスト | 自ライフ → ダスト |
| 焦燥ダメージ | 自オーラ or 自ライフ → ダスト |
| 付与結晶減少 | 付与札 → ダスト |
| 切札使用 | 自フレア → 使用済(カード上/ダスト) |

> **重要**: ライフで受けたダメージはフレアへ行く。これにより、ダメージを受けることが切札のコストを蓄積する行為にもなる。この「受動的フレア蓄積」がゲームの戦略的ジレンマの核心。

---

## 5. オーラ上限ルール

- オーラは最大**5**。
- オーラが5の状態で結晶をオーラに移動する効果は「できない」として処理される。
- 前進時にオーラが5なら前進不可。纏い時にオーラが5なら纏い不可。
