# キーワード能力の厳密な定義

各メガミの固有キーワードおよび共通キーワードの**発動条件と効果**を網羅する。
カードテキストに「〇〇であるならば」「〇〇のとき」と書かれた条件はここで定義される。

---

## 1. 状態参照キーワード（条件を満たしている間、効果が有効）

### 決死（けっし）
- **条件**: 自分のライフが **3以下**
- **効果**: カードに記載された決死専用の追加効果・ダメージ強化が有効になる
- **頻出例**: ユリナの「居合」は決死時に4/3にダメージ増加
- **戦術的意味**: ライフ3以下は自分の攻撃力が上がるが、同時に相手のリーサル圏内でもある

### 八相（はっそう）
- **条件**: 自分のオーラが **1以下**
- **効果**: カードに記載された八相専用の効果が有効になる
- **戦術的意味**: 防御を捨てて攻撃力を得るハイリスク状態

### 境地（きょうち）
- **条件**: 自分の集中力が **2**（最大値）
- **効果**: カードに記載された境地専用の効果が有効になる
- **戦術的意味**: 集中力を基本動作に消費しない＝手札を温存している状態
- **主なメガミ**: トコヨの多くのカードが境地条件を持つ

### 灰塵（はいじん）
- **条件**: ダストが **12以上**
- **効果**: カードに記載された灰塵専用の効果が有効になる
- **戦術的意味**: ゲーム終盤に自然到達するが、意図的にダストを増やして早期達成も可能
- **主なメガミ**: ウツロ、ハガネ

### 鏡映（きょうえい）
- **条件**: 特定の数値が**相手と同じ**（オーラ同値、ライフ同値、間合と特定の値が同値、など）
- **効果**: ヤツハのカードに記載された鏡映専用の効果が有効になる
- **主なメガミ**: ヤツハ

## 2. 使用条件キーワード（カード使用時に参照）

### 連火（れんか）
- **条件**: そのターンに使用したカードが**3枚目以降**
- **効果**: 連火条件付きの追加効果が発動する
- **カウント**: カードの「使用」をカウント（基本動作はカウントしない）
- **例**: 1枚目→2枚目→**3枚目から連火**

### 設置（せっち）
- **条件**: カードが**伏せ札の状態から使用**される
- **効果**: 設置条件を持つカードは伏せ札から使用することで追加効果を得る
- **主なメガミ**: オボロ
- **重要**: 手札から使用した場合は設置条件を満たさない

### 全力（ぜんりょく）
- **条件**: メインフェイズで**全力行動を選択して使用**
- **制約**: 全力カード使用後、そのターンは他の行動不可。全力カード使用前に標準行動を行っていた場合も使用不可。

### 対応（たいおう）
- **条件**: 相手の攻撃が宣言された後の**対応ウィンドウ**で使用可能
- **効果**: 相手の攻撃に割り込んで効果を先に解決する
- **詳細**: `03_card_resolution.md` のStep 2-3を参照

## 3. メガミ固有キーワード

### 開閉（かいへい）— ユキヒ専用
- **仕組み**: ユキヒは「傘」の状態を持つ。**開（開傘）**と**閉（閉傘）**の2状態。
- **効果**: カードのテキストが傘の状態によって変化する（開閉で別効果）。
- **切り替え**: 特定のカード効果やターンの進行で傘の状態を切り替える。

### 計略（けいりゃく）— オボロ関連
- **仕組み**: 特定の条件（カード効果やフェイズ処理）で発動する謀略的効果。
- **効果**: カードテキストに定義される。

### 遠心（えんしん）— ハガネ専用
- **条件**: そのターン中に間合が**2以上変動**した後
- **効果**: 遠心条件を持つカード・効果が有効化される
- **計測**: 合計変動量ではなく、1回の変動もしくは累計変動で2以上

### 騎動（きどう）— サリヤ専用
- **仕組み**: サリヤの「造花結晶」を移動させるアクション
- **造花結晶**: サリヤが管理する独自の結晶。燃焼して攻撃力バフや移動に使う。
- **効果**: 騎動によって間合を変動させたり、攻撃力を上げたりする。

### 凍結（とうけつ）— コルヌ専用
- **仕組み**: 特定のエリアの結晶に「凍結トークン」を載せ、移動を制限する
- **効果**: 凍結された結晶は通常の移動（基本動作など）では動かせない

### 毒（どく）— チカゲ専用
- **仕組み**: 相手の手札に「毒カード」を送り込む
- **毒カードの効果**: 手札を圧迫し、有効なカードの枚数を減らす。毒カードの種類ごとに異なるデメリットがある。
- **種類**: 弛緩毒、幻覚毒、滅灯毒 等

### 機巧（きこう）— クルル専用
- **条件**: そのターン中に使用した**カードのサブタイプの組み合わせ**を参照する
- **例**: 「機巧〈攻撃, 攻撃〉」= そのターン中に攻撃カードを2枚以上使用済みの場合に発動
- **重要**: サブタイプを参照するため、他メガミのカードも条件に含まれる

### 萎縮（いしゅく）— ウツロ関連
- **公式用語**: 「萎縮」が正式表記。「畏縮」は誤記・表記揺れのため使用しないこと。
- **効果を受けた側への影響**: 次の開始フェイズで集中力が**増加しない**
- **持続**: 1回の開始フェイズ分。毎ターン付与し続ければ毎ターン無効化できる。

### 帯電（たいでん）— ライラ専用
- **仕組み**: ライラ固有のゲージ。カードを使用するたびに帯電ゲージが変動する。
- **効果**: ゲージの状態に応じてカードの攻撃力・効果が変化する。

### 守護（しゅご）— ホノカ専用
- **仕組み**: ホノカは「精霊」カードを持ち、精霊を「開花」させることで強化していく。
- **効果**: 精霊が成長することで、後半に向けて強力な効果が解放される。

### 終焉影（しゅうえんえい）— ウツロ専用
- **仕組み**: ウツロの変身形態。特定条件で「終焉影」に変身する。
- **効果**: 変身後はカード性能が変化し、より破壊的な能力を得る。

※ 「畏縮」は「萎縮」の誤表記。上記「萎縮」の項を参照。

## 4. 戦術キーワード（ルール上の用語ではなくコミュニティ用語）

| 用語 | 意味 |
|:---|:---|
| **レンジロック** | 特定の間合を維持し続け、相手に有利な間合で攻撃させない戦術 |
| **リーサル** | 相手を倒しきれるダメージ計算が成立する状態 |
| **グッドスタッフ** | 単体性能の高いカードを組み合わせたデッキ |
| **ビートダウン** | 攻撃を軸にライフを削る戦術 |
| **コントロール** | 相手のリソースや行動を制限して優位に立つ戦術 |
| **テンポ** | 手数やリソースの効率で相手を上回ること |
| **裏飛沫（うらしぶき）** | ウツロの特定コンボ条件を達成すること |
