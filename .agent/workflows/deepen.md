# /deepen — 戦略深化

`furuyoni_strategist` スキルを用いて、メガミ・ペアの戦略記述を分析し、具体性・正確性を強化する。

## 手順

1. `.agent/skills/furuyoni_strategist/SKILL.md` を読み込む。

2. 対象ファイルを特定する:
   - 引数でメガミ名/ファイルパスが指定されていればそのファイル
   - 未指定なら `docs/megami/*.md` の全ファイル + `docs/pairs/index.md`

3. 各ファイルから以下のセクションを抽出する:
   - 「本質的な解説」
   - 「戦術の核心」
   - 「おすすめの組み合わせ」

4. **Sensei ペルソナ** でレビュー:
   - 初心者が「なぜこのメガミ/ペアが強いか」を5秒で理解できるか
   - 抽象的すぎる記述（「強い」「便利」）がないか
   - 具体的なカード名・数値が含まれているか

5. **Meijin ペルソナ** でレビュー:
   - 戦略記述がシーズン9/10のメタに即しているか
   - シナジーの主張が実際の対戦で再現可能か
   - 見落とされている重要なカウンタープレイがないか
   - おすすめの組み合わせの勝ち筋が明確か

6. 必要に応じて外部情報を参照:
   - 公式サイト (`https://main-bakafire.ssl-lolipop.jp/furuyoni/na/`)
   - `docs/community/blogs.md` に記載のブログ記事
   - `docs/strategy/meta.md` の環境レポート

7. 改善案を `notify_user` でユーザーに提示する。

8. 承認後、修正を適用する。
