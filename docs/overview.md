# 桜降る代に決闘を (ふるよに) - ゲーム概要

桜降る代に決闘を（通称：ふるよに）は、BakaFire Partyによってデザインされた2人用対戦型カードゲームです。

## ゲームの本質

プレイヤーは「ミコト」となり、メガミの力を[宿し](mechanics/index.md)て戦います。
一般的なトレーディングカードゲーム（TCG）とは異なり、事前に構築したデッキを持ち込むのではなく、「[眼前構築](mechanics/flow.md)」という独自のシステムを採用しています。

### 主な特徴

1.  2柱のメガミを選択: 多数のメガミの中から2柱を選びます（「[双掌繚乱](mechanics/flow.md)」ルールなど）。
2.  相手を見てからデッキを組む: 対戦相手が選んだ2柱のメガミを確認してから、デッキ構築を始めます。
3.  その場でデッキ構築: 自分選んだ2柱のメガミのカードプール（[通常札](mechanics/index.md)14枚、[切札](mechanics/index.md)8枚）から、[通常札](mechanics/index.md)7枚と[切札](mechanics/index.md)3枚を選んでデッキを組みます。
4.  [間合](mechanics/index.md)（まあい）の概念: お互いの距離（[間合](mechanics/index.md)）ボード上で管理します。この「[間合](mechanics/index.md)」のコントロールが勝敗を分けます。

## 世界観

桜が絶えず舞い散る世界。この桜は[桜花結晶](mechanics/index.md)と呼ばれ、不思議な力を秘めています。
ミコトたちはこの結晶を操り、身を守る[オーラ](mechanics/index.md)としたり、強力な技（[フレア](mechanics/index.md)）の糧としたり、あるいは二人の間の距離（[間合](mechanics/index.md)）そのものとして扱います。

## なぜこの「ペアガイド」が必要なのか？

メガミの組み合わせ（ペア）は100通り以上存在し、それぞれ全く異なる戦略が必要になります。
初心者にとって、この膨大な可能性は時にハードルとなります。
このドキュメントとアプリケーションは、以下の目的のために作られました：

1.  自分に合ったペアを見つける: 自分のプレイスタイルに合った組み合わせを探す手助けをします。
2.  基本を学ぶ: 厳選された「ストーリー」を通じて、ゲームの基本概念を段階的に習得します。
3.  知識を深める: 各メガミの強み、弱み、キーカードを理解します。
