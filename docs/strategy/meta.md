# シーズン10 メタレポート

## 環境概要 (2025年 - S10期)

現在進行中のシーズン10（S10）は、カード更新によりバランス調整が行われた環境です。

### 注目される変更点
これまでのカード更新（S9-2からS10への移行など）により、以下のメガミに調整が入っています。
*   **ミズキ (兜)**: 防御性能の調整（一部カードの修正）。
*   **マヒル (現在)**: 攻撃性能の調整。
*   **ハガネ (槌) / サリヤ (騎)**: 過去のシーズンで猛威を振るったため、継続的な調整対象となっています。
