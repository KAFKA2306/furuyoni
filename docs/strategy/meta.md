# シーズン10 メタレポート

## 環境概要 (2026年 - S10-2期)

現在は2026年1月のカード更新を経て、**シーズン10-2 (S10-2)** が進行中です。

### 注目される変更点 (2026年1月-2月)
*   **最新カード更新**: 2026年1月に、S10からS10-2への大規模なバランス調整が行われました。
*   **禁止改定**: 2026年1月および2月に最新の禁止改定が実施されています。
    *   **レッドバレット**: [ヒミカ](../megami/index.md)（銃）/ [ウツロ](../megami/index.md)（鎌）の組み合わせにおいて、引き続き[眼前構築](../mechanics/glossary.md)での選択が禁止されています。
*   **VR版の展開**: VRChatでのオープンベータテストが進行中であり、コミュニティのアクティビティが非常に高まっています。
