# ゲームプランと勝利の勘所

『桜降る代に決闘を』（ふるよに）において、どのように勝ち筋を描き、実行に移すのか。ルールとカード効果に基づいた戦略の核心を解説します。

## 1. 勝利の定義：[ライフ](../mechanics/glossary.md)を0にする

このゲームの唯一の目標は、相手の**[ライフ](../mechanics/glossary.md)を0**にすることです。
しかし、漫然と攻撃を振るだけでは勝てません。[ライフ](../mechanics/glossary.md)を減らすには以下の2つのルートを意識する必要があります。

-   **攻撃による[ライフ](../mechanics/glossary.md)ダメージ**: 攻撃カードの右側の数値（X/**Y**）で[ライフ](../mechanics/glossary.md)を削る。
-   **[再構成](../mechanics/glossary.md)ダメージ**: 相手が山札を引き切り、[再構成](../mechanics/index.md)を行うたびに受ける1ダメージ。

### 勘所：リーサル計算
相手の[ライフ](../mechanics/glossary.md)が残り3〜4点になった瞬間が「勝負所」です。自分の手札、[集中力](../mechanics/glossary.md)、そして[フレア](../mechanics/glossary.md)（[切札](../mechanics/glossary.md)）で、今この瞬間に[ライフ](../mechanics/glossary.md)を削りきれるか（リーサル）を常に計算してください。

## 2. 資源の循環：[桜花結晶](../mechanics/glossary.md)の法則

ふるよにの戦略の根幹は、**「[桜花結晶](../mechanics/glossary.md)がどこからどこへ動くか」**を制御することにあります。

### リソース・ループ
1.  **[オーラ](../mechanics/glossary.md)を削る**: 攻撃を[オーラ](../mechanics/glossary.md)で受けさせ、結晶を[ダスト](../mechanics/glossary.md)へ送る。
2.  **[ライフ](../mechanics/glossary.md)を削る**: [オーラ](../mechanics/glossary.md)がない、あるいは防ぎきれない攻撃を[ライフ](../mechanics/glossary.md)で受けさせ、結晶を[フレア](../mechanics/glossary.md)へ送る。
3.  **[フレア](../mechanics/glossary.md)を活用する**: [ライフ](../mechanics/glossary.md)受けで得た[フレア](../mechanics/glossary.md)をコストに、強力な**[切札](../mechanics/glossary.md)**を放つ。

### 勘所：ダメージ選択のジレンマ
攻撃を受けた際、「[オーラ](../mechanics/glossary.md)で受けるか[ライフ](../mechanics/glossary.md)で受けるか」の選択が勝敗を分けます。
-   **[オーラ](../mechanics/glossary.md)受け**: 次の攻撃を防ぐ盾を維持するが、[フレア](../mechanics/glossary.md)は増えない。
-   **[ライフ](../mechanics/glossary.md)受け**: [フレア](../mechanics/glossary.md)が増えて[切札](../mechanics/glossary.md)が使えるようになるが、[ライフ](../mechanics/glossary.md)が減り、敗北に近づく。

> [!IMPORTANT]
> **「あえて[ライフ](../mechanics/glossary.md)で受けて、返しのターンに[切札](../mechanics/glossary.md)で仕留める」**という判断が、中級者への第一歩です。

## 3. 距離の支配：[間合](../mechanics/glossary.md)の管理

[間合](../mechanics/glossary.md)は単なる数字ではなく、**そのメガミの「射程圏」**を表します。

-   **得意[間合](../mechanics/glossary.md)を維持する**: 自分の強力な攻撃カードが使える距離（例：[ユリナ](../megami/index.md)なら3-4、[ヒミカ](../megami/index.md)なら5-10）をキープします。
-   **相手の得意[間合](../mechanics/glossary.md)から逃げる**: 相手が攻撃できない距離へ、[基本動作](../mechanics/glossary.md)（[後退](../mechanics/index.md)・[纏い](../mechanics/index.md)・[前進](../mechanics/index.md)）で移動します。

## 4. ゲームの進行：3つのフェイズ

### 序盤：準備と牽制（1〜2巡目）
-   **[宿し](../mechanics/index.md)**で[フレア](../mechanics/glossary.md)を最低限確保する。
-   相手の[オーラ](../mechanics/glossary.md)を適度に削り、[ダスト](../mechanics/glossary.md)を作る（自分の**[纏い](../mechanics/index.md)**のため）。
-   手札を整え、強力なコンボが打てる準備をする。

### 中盤：リソースの削り合い（[ライフ](../mechanics/glossary.md) 7〜4）
-   互いに[ライフ](../mechanics/glossary.md)を削り、[切札](../mechanics/glossary.md)が使用可能になる。
-   [対応](../mechanics/index.md)カードを構え、相手の大技を警戒する。

### 終盤：リーサルの攻防（[ライフ](../mechanics/glossary.md) 3以下）
-   [オーラ](../mechanics/glossary.md)を5（上限）に保ち、[ライフ](../mechanics/glossary.md)への致命傷を防ぐ。
-   **[再構成](../mechanics/glossary.md)ダメージ**を考慮し、あえて[再構成](../mechanics/glossary.md)せずに戦う判断も必要。
-   リソースを全て使い切り、相手の[ライフ](../mechanics/glossary.md)を0に叩き込む。

## 5. 勝利のためのチェックリスト

- [ ] 相手の最大打点（どの[切札](../mechanics/glossary.md)で何点飛んでくるか）を把握しているか？
- [ ] 自分の次のターンのドロー枚数を考慮して手札を調整したか？
- [ ] 無駄な「[前進](../mechanics/index.md)」で、相手が得意な近距離に飛び込んでいないか？
- [ ] [フレア](../mechanics/glossary.md)は足りているか？ [宿し](../mechanics/index.md)すぎで[オーラ](../mechanics/glossary.md)がスカスカになっていないか？

このゲームプランを意識することで、あなたの決闘は「運」から「戦略」へと進化します。
