# 版・フォーマット統合ガイド

このサイトは、現行シリーズ**『桜降る代に決闘を 再演』**と、旧シリーズ**『新幕 桜降る代に決闘を』**の資料を同じ入口から参照できるように整理します。

## まず遊ぶフォーマットを選ぶ

| フォーマット | 主な対象 | このサイトでの扱い |
| --- | --- | --- |
| 再演起源戦 | 再演の基本的なカードプール | 公式ルール・カード情報へ案内 |
| 再演完全戦 | 再演で利用できる広いカードプール | 公式ルール・カード情報へ案内 |
| 再演古典戦 | 新幕を含む従来資産を活用する形式 | 再演公式情報と新幕アーカイブを横断 |
| 新幕シーズン1〜10 | 過去環境の調査・振り返り | 歴史アーカイブとして保存 |

- [再演 公式サイト](https://furuyoni.sekiseiro.com/re/)
- [再演 公式ルール](https://furuyoni.sekiseiro.com/re/rules/)
- [再演 総合ルール](https://furuyoni.sekiseiro.com/re/rules/comprehensive-rule/)
- [新幕 旧公式サイト](https://main-bakafire.ssl-lolipop.jp/furuyoni/)

再演の公式総合ルールには「2026/06/03 公開・微調整」と記載されています。新幕公式サイトは更新を終了し、2026年5月以降の情報は再演公式サイトへ移行しています。

## 情報ラベル

| 表示 | 意味 |
| --- | --- |
| 再演・公式 | 現行の公式ページを直接参照 |
| 再演・非公式整理 | 再演の公式情報を探しやすく再構成 |
| 新幕・公式アーカイブ | 旧公式サイトや公式資料に基づく記録 |
| 新幕・非公式解説 | 管理者またはコミュニティによる解釈 |
| 版共通 | 再演・新幕の双方で確認して使う基礎概念 |
| 検証待ち | 一次出典との対応が未確認 |

## 統合方針

1. 旧版資料を削除せず、再演のフォーマット選択から到達できるようにする。
2. すべての戦術・カード・裁定情報に対象版を示す。
3. 「現在」「最新」は再演公式情報だけに使用する。
4. 新幕のTier、構築例、裁定、世界観解釈は、公式事実と非公式解説を分離する。
5. 同名カードや同じメガミでも、再演と新幕で差がある前提で扱う。
6. 正確な裁定が必要な場面では、公式ルール・FAQ・大会審判を優先する。

## このサイトの役割

このサイトは、再演だけに置き換えるサイトでも、新幕だけを保存するサイトでもありません。

> **再演の入口と、新幕の蓄積を、版を混同せずに接続する非公式統合索引**

として運用します。
