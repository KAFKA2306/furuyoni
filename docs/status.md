# このサイトの版・情報方針

## 現行版

- [再演 公式サイト](https://furuyoni.sekiseiro.com/re/)

## このサイトに残る旧版資料

メガミ個別ページ、カード画像、シーズン1〜10の資料の多くは、旧シリーズ**「新幕」**を対象としています。

旧版資料には、次の表示を付けます。

| 表示 | 意味 |
| --- | --- |
| 現行版 | 再演の公式情報に基づく案内 |
| 新幕アーカイブ | 旧シリーズの記録。再演へ直接適用しない |
| 非公式解説 | 管理者またはコミュニティによる解釈 |
| 検証中 | 一次出典との対応を確認できていないため、内容を案内しない |

## 掲載方針

1. 現行ルール、カード、製品、イベントは公式ページへ案内する。
2. 版が不明な情報を「現在」「最新」と表現しない。
3. Tier、最強構築、裁定、世界観要約は、一次出典を示せない場合は掲載しない。
4. 旧版資料は削除せず、アーカイブであることを明示する。
5. 外部コミュニティ記事は、公式情報と混同しない。
