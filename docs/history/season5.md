# シーズン5

## 期間
2019年11月6日 ～ 2020年9月

## 概要
シーズン5は、第4拡張の発売と共に始まり、長期にわたって続いたシーズンです。環境の変遷に合わせて、5.0、5.1、5.2というサブシーズンに区分されます。この時期にゲームのバランス調整方針が大きく確立されました。

## サブシーズンの変遷

### シーズン5.0 (2019/11/06 - 2020/01/05)
- 特徴: 禁止カードなしのスタート。
- 環境: 「忍嵐」（[オボロ](../megami/index.md)/[ライラ](../megami/index.md)）が優勢なデッキとして環境をリードしました。

### シーズン5.1 (2020/01/06 - 2020/05/03)
- 変更点:  カード「流転爪」が禁止指定を受けました。
- 環境: 禁止改定により「忍」が弱体化。代わって「騎嵐」（[トコヨ](../megami/index.md)/[ライラ](../megami/index.md)）が圧倒的なシェアを獲得し、環境を支配しました。

### シーズン5.2 (2020/05/04 - 2020/09/10)
- 変更点: 追加で「ASURA」（変身カード）および「暴風」が禁止カードに指定されました。
- 環境: 「戦新嵐」が開拓され、「新戦鏡」や「銃騎嵐」などがメタゲームに登場。環境は「三拾一捨」形式での戦略が深化しました。

## 代表的な出来事・メタ
- 禁止改定のインパクト: シーズンを通して「流転爪」や「ASURA」といったパワーカードが禁止されることで、環境が強制的に変化させられました。
- ASURAの暴走と終焉: 強力すぎた「Form:ASURA」は、シーズン6での修正を待たずして禁止カードとなりました。
