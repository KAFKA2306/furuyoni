# シーズン9 & 9-2

## 期間
シーズン9開始: 第九拡張『新劇拡張後編』発売 (2024年〜)
シーズン9-2: 2024年6月24日以降の大規模カード更新後

## 概要
シーズン9では、新メガミ（弓、電子）が登場し、環境が大きく変動しました。続くシーズン9-2では、強力すぎるカードに対する衝撃的な改定が行われ、メタゲームが再構築されました。

## 環境の特徴

### シーズン9: 新世代の到来
- **新メガミの席巻**: 新たに追加されたメガミ（特に弓、電子）が環境を支配し、メタゲームの中心となりました。
    *   **弓 (Yumi)**: 遠距離からの高火力アタッカー。
    *   **電子 (Electronic)**: 特殊リソース「電圧」を管理し、爆発的なアドバンテージを得るテクニカルなメガミ。
- **アキナの強さ**: シーズン8から引き続き**アキナ**が強力で、環境のトップメタの一角を占め続けました。

### シーズン9-2 (2024.06〜)
この期間は「ゲームバランスが完成した」と評される一方で、競技的には明確なTierが存在しました。

- **ハガネ (リスペクト) のナーフ**: 「リスペクト」が強力すぎたため、早期に下方修正されました。
    ![ハガネ](../assets/images/chara/ico-08.png)
- **注目デッキ**:
    *   **鎌爆誕 (
        ![ウツロ](../assets/images/chara/ico-13.png) **ウツロ** / 
        ![ホノカ](../assets/images/chara/ico-14.png) **ホノカ**
        )**: 特定のコンボにより爆発的な火力を発揮。
    *   **騎旗算 (
        ![トコヨ](../assets/images/chara/ico-04.png) **トコヨ** / 
        ![ホノカ](../assets/images/chara/ico-14.png) **ホノカ** / 
        ![シンラ](../assets/images/chara/ico-07.png) **シンラ**
        )**: 三拾一捨形式で強力な組み合わせ。特にシンラ（算）は「完全論破」などのコントロール性能で環境を定義しました。
    *   **遺物 (経典)**: 「使徒」との組み合わせで高い瞬間火力を出す「遺物算」なども研究されました。
