   [シンラ](../megami/index.md)は「相手のデッキギミック破壊」「悠長なプランの否定」「レンジロック補助・対抗」など、“ふるよにの基本構造から外れた勝ち筋”を取り締まるのが強みで、相方の火力補助やサブ火力も担えます。薙書のように、相方でビートもコントロールもカバーできる組み合わせはデッキタイプの射程が広く、対面適応力が高いのが売りです。([へくとぱ雑文投棄所][9])
