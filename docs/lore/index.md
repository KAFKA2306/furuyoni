# 桜降る代の世界（Lore）

このゲームの背景にある、美しくも残酷な「桜降る代世界」の物語を紹介します。

## 世界背景

### 桜花結晶（おうかけっしょう）
この世界で舞い散る桜の花びらは、単なる植物ではなく「桜花結晶」と呼ばれるエネルギーの結晶体です。
これは人々の生活を支える動力源であり、貨幣代わりであり、そしてメガミの力の源でもあります。

### ミコト（命/尊）
桜花結晶の力を操り、メガミの力を身に宿して戦う能力者のこと。
すなわち、プレイヤーであるあなた自身です。
ミコトはメガミと契約し、その権能（カード）を借り受けることで、常人離れした力を発揮します。

### 桜花決闘（おうかけっとう）
ミコト同士がメガミの力を借りて行う儀式的な決闘。
それは単なる争いではなく、神聖な意味を持つ場合もあれば、政治的な解決手段として用いられる場合もあります。

## 主要な物語

### はじまりの神話
かつて世界は混沌としていました。
最初のメガミたちが現れ、月を割り、海を干上がらせ、大地を作りました。
特にユリナとサイネの因縁は深く、彼女たちの戦いが現在の桜降る代の在り方を決定づけたと言われています。

### 英雄の叙事詩
時代は下り、数々のミコトとメガミの物語が紡がれました。

- 天音揺波（あまね ゆりな）: 武神・ユリナの力を宿し、数々の伝説を残した最強のミコト。彼女の歩んだ道が、多くのメガミたちの運命を交差させました。
- 氷雨細音（ひさめ さいね）: 揺波のライバルであり、親友。冷静沈着な戦いで揺波を支え、時に立ちはだかりました。

## メガミとは何か？

メガミは、概念や事象が具現化した存在です。

- オリジン: そのメガミの「原型」となった存在や物語。
- アナザー: 異なる側面や、もしもの可能性（IF）を具現化した姿。例えば「ユリナ」に対する「Aユリナ（古刀）」など。
