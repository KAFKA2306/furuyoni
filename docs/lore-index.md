## 新幕・公式アーカイブ

## このサイトの旧世界観メモ

既存の世界観ページには、公式記述と管理者による要約・解釈の境界が明示されていません。内容自体は削除せず、検証前のアーカイブとして残します。

[旧世界観メモを開く](lore.md){ .md-button }

!!! warning "旧世界観メモの注意"
    「交易の貨幣」「八葉鏡の儀式」「三座」などの説明には、ページ内で一次出典が示されていません。公式設定として引用せず、公式物語との照合が完了するまで非公式メモとして扱ってください。

## 今後の統合方法

世界観情報は、次の単位で再構成します。

1. 公式ページまたは公式小説の出典。
2. 対象シリーズ・時期。
3. 公式本文から確認できる事実。
4. 管理者による要約・解釈。
5. ネタバレ範囲。
