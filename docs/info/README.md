# ふるよにナレッジベース

**桜降る代に決闘を（ふるよに）** のための包括的なガイドへようこそ。
このディレクトリには、ゲームのルール、キャラクター詳細、および戦略に関する情報が含まれています。

## ドキュメント一覧

**1. はじめに**
*   [00_overview.md](./00_overview.md): ふるよにとは？ このガイドの目的。

**2. ゲームルール**
*   [01_mechanics.md](./01_mechanics.md): 基本用語（間合、オーラ、フレアなど）とキーワード能力の解説。
*   [03_game_flow.md](./03_game_flow.md): 準備（双掌繚乱/三拾一捨）から決着までのゲームの流れ。

**3. メガミデータ**
*   [02_megami_roster.md](./02_megami_roster.md): 全16柱のメガミの権能、戦術、キーワード解説。

**4. 学習カリキュラム**
*   [04_beginner_curriculum.md](./04_beginner_curriculum.md): 初心者がステップバイステップで学べる、全8章のストーリーモード（推奨ペア）。

**5. コミュニティリソース**
*   [pair.md](./pair.md): 詳細なペア解説と分析（既存ファイル）。
*   [blogs.md](./blogs.md): Advent Calendar記事のアーカイブ（既存ファイル）。

## 使用方法
これらのファイルは「ふるよにペアガイド」アプリケーションのデータソースとして意図されていますが、単独で戦略Wikiとして読むことも可能です。
