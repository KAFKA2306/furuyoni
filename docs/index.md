## どの形式で遊びますか

### 再演起源戦・再演完全戦

現行のルール、カード、禁止改定は再演公式サイトを基準にします。

[再演 公式ルール](https://furuyoni.sekiseiro.com/re/rules/){ .md-button .md-button--primary }

## 情報の読み方

同じメガミ名・カード名でも、再演と新幕でカード内容や使用可能範囲が異なる場合があります。ページ上の版表示と、公式カード情報を確認してください。

このサイトは非公式です。正確な裁定、使用可能カード、大会情報は公式案内を優先します。
