# ふるよに基礎知識：決闘の前に知っておくべきこと

「桜降る代に決闘を（ふるよに）」の世界へようこそ。
このページでは、ルール以前の**「このゲームがどういう仕組みで動いているのか」**という本質的な基礎知識を、専門用語を抑えて解説します。

## 1. 全ては「[桜花結晶](glossary.md)」の循環である

このゲームの最大の特徴は、ボード上のあらゆるところにあるピンク色の石**「[桜花結晶](glossary.md)」**です。
体力も、防御も、エネルギーも、距離も、すべてはこの結晶の数で表されます。

結晶は「消える」ことはほとんどなく、プレイヤーの行動によって板の上を**ぐるぐると循環**します。

### 5つのエリア
結晶が置かれている場所によって、その意味が変わります。

1.  **[間合](glossary.md)（距離）**: 二人の間の距離。ここにある結晶を自分の手元に引き寄せると「距離が縮まる（[前進](index.md)）」ことになります。
2.  **[オーラ](glossary.md)（盾）**: あなたを守るバリア。最大5つまで。
3.  **[ライフ](glossary.md)（体力）**: これが0になると負けです。
4.  **[フレア](glossary.md)（エネルギー）**: 強力な「[切札](glossary.md)」を使うためのゲージ。ダメージを受けたり、自分の意思で溜めたりします。
5.  **[ダスト](glossary.md)（空き地）**: 誰のものでもない結晶。コストとして支払われた結晶はここに行きます。

> [!TIP]
> **「自分の[オーラ](glossary.md)を[フレア](glossary.md)に送って必殺技の準備をするか？」「今は[オーラ](glossary.md)を5個貯めて守りを固めるか？」**
> この「結晶をどこに動かすか」の判断こそが、ふるよにの面白さの核です。

## 2. カードの見方：直感で理解する

カードには数字やアイコンが並んでいますが、覚えるべきは3つだけです。

### 攻撃範囲（[間合](glossary.md)）
カードの左上にある数字（例：`3-4`）は、**「相手との距離が3か4の時だけ使える」**という意味です。
相手が遠すぎても近すぎても攻撃は当たりません。

### ダメージ（[オーラ](glossary.md)/[ライフ](glossary.md)）
攻撃カードには `2/1` のように数字が書かれています。

*   **左側の数字（[オーラ](glossary.md)ダメージ）**: 相手の「盾」をどれだけ削るか。
*   **右側の数字（[ライフ](glossary.md)ダメージ）**: 相手の「体力」をどれだけ削るか。

> [!IMPORTANT]
> 相手は攻撃を受けた時、**「盾（[オーラ](glossary.md)）で受けるか、体力（[ライフ](glossary.md)）で受けるか」を選ぶことができます。**
> 最初は盾で防がれますが、盾を削りきれば、本命の体力を削るチャンスが生まれます。

## 3. 「[眼前構築](glossary.md)」：相手を見てから武器を選ぶ

一般的なカードゲームは、家でデッキを組んで持ってきますが、ふるよには違います。

1.  **メガミ（武器）を2人選ぶ。**
2.  **相手の選んだ2人を見る。**
3.  **その場で、自分の持っているカードから10枚選んでデッキを作る。**

「相手が足の速い忍者なら、こちらは待ち伏せの罠を多めにしよう」といった、対戦相手に合わせた柔軟な戦略が、戦う前から始まっています。
