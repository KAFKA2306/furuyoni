# ふるよにの基本メカニクスと用語

## コアコンセプト

ふるよにをプレイする上で最も重要なのは、ボード上の結晶の動きを理解することです。
まずは以下の「基礎知識」を一読することをお勧めします。

### 1. リソース（[桜花結晶](glossary.md)）
ゲーム内のほぼ全てのリソースは、[桜花結晶](glossary.md)というトークンで表されます。
トークンが置かれている場所によって、その意味が変わります。

*   [ライフ](glossary.md): プレイヤーの体力。通常は10です。（初心者用ルール等では8の場合もあります）。0になると敗北します。
*   [オーラ](glossary.md): プレイヤーを守る盾。キャラクターカードの上に置かれます。ダメージを受ける際、[ライフ](glossary.md)の代わりに[オーラ](glossary.md)で受けることができます（**上限5**）。
*   [フレア](glossary.md): [切札](glossary.md)を使用するためのコスト（エネルギー）。攻撃を[オーラ](glossary.md)で受けた際や「[宿し](glossary.md#yadoshi)」によって溜まります。
*   [間合](glossary.md)（まあい）: プレイヤー間の**物理的な距離**。ボード中央にある結晶の数で表されます。基本の開始距離は10です。
*   [ダスト](glossary.md): ボード中央の共有エリアにある結晶。どのプレイヤーのリソースでもない状態を表します。様々なコストや効果で移動します。

### 2. カードの種類
主に以下の種類のカードがあります。

*   [通常札](glossary.md): 緑色の枠。デッキに7枚入れます。山札から引いて手札として使用します。
*   [切札](glossary.md): 赤色の枠。デッキに3枚入れます。裏向きで「[切札](glossary.md)置き場」に置かれ、[フレア](glossary.md)を支払うことで使用条件を満たせばいつでも使用できます。使用後は通常、表向き（使用済み）になります。

### 3. ダメージの仕組み
攻撃カードには「2/1」のようにダメージが記載されています。
*   **左側の数値（[オーラ](glossary.md)ダメージ）**: 相手が[オーラ](glossary.md)で受ける場合に減少する結晶の数。[オーラ](glossary.md)以上のダメージを受けても、[ライフ](glossary.md)には貫通しません。
*   **右側の数値（[ライフ](glossary.md)ダメージ）**: 相手が[ライフ](glossary.md)で受ける場合に減少する結晶の数。[ライフ](glossary.md)が0になると敗北します。
*   **ダメージの選択**: 「[オーラ](glossary.md)へのダメージ」以上の[オーラ](glossary.md)がある場合のみ、[オーラ](glossary.md)による防御を選択できます。足りない場合は必ず[ライフ](glossary.md)で受けます。（対応不可や特定の効果で固定される場合を除く）。

### 4. [基本動作](glossary.md)
自分のターン中、1「[集中力](glossary.md)」を使うか、手札1枚を「伏せ札（捨て札）」にすることで、以下の動作を行えます。

1.  [前進](glossary.md#advance): [間合](glossary.md)から[オーラ](glossary.md)へ結晶を1つ動かす。（[間合](glossary.md)が0より大きく、[オーラ](glossary.md)が上限未満の場合）。距離を詰めつつ防御を固めます。※[達人の間合](glossary.md)（2以下）では行えません。
2.  [後退](glossary.md#retreat): [オーラ](glossary.md)から[間合](glossary.md)へ結晶を1つ動かす。（[オーラ](glossary.md)がある場合）。距離を取りますが、防御が薄くなります。
3.  [纏い](glossary.md#matoy)（まとい）: [ダスト](glossary.md)から[オーラ](glossary.md)へ結晶を1つ動かす。防御を回復します。
4.  [宿し](glossary.md#yadoshi)（やどし）: [オーラ](glossary.md)から[フレア](glossary.md)へ結晶を1つ動かす。[切札](glossary.md)のためのゲージを溜めます。

## ターンの流れ

1.  [開始フェイズ](flow.md):
    *   [集中力](glossary.md)を1得ます（上限2）。ただし「畏縮」状態の場合は、[集中力](glossary.md)を得る代わりに「畏縮」を解除します。
    *   [再構成](flow.md): 山札が空の場合、または任意で宣言した場合、捨て札を山札に戻してシャッフルします。この時、[ライフ](glossary.md)に1ダメージを受けます（[再構成](flow.md)ダメージ）。
    *   カードを2枚引く（先攻1ターン目は3枚手札から開始し、ドローなし等、細かいルールあり）。

2.  [メインフェイズ](flow.md):
    *   [基本動作](glossary.md)を行う。
    *   手札からカードを使用する。
    *   [切札](glossary.md)を使用する。

3.  [終了フェイズ](flow.md):
    *   手札上限（通常は2枚）になるようにカードを捨てる。
    *   「ターン終了時」の効果を処理する。
