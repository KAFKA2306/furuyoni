# ふるよにの基本メカニクスと用語

## コアコンセプト

ふるよにをプレイする上で欠かせない用語の解説です。

### 1. リソース（桜花結晶）
ゲーム内のほぼ全てのリソースは、桜花結晶というトークンで表されます。
トークンが置かれている場所によって、その意味が変わります。

*   ライフ: プレイヤーの体力。通常は10です。（初心者用ルール等では8の場合もあります）。0になると敗北します。
*   オーラ: プレイヤーを守る盾。キャラクターカードの上に置かれます。ダメージを受ける際、ライフの代わりにオーラで受けることができます（上限5）。
*   フレア: 切札を使用するためのコスト（エネルギー）。オーラへのダメージや「宿し」によって溜まります。
*   間合（まあい）: プレイヤー間の物理的な距離。ボード中央に置かれます。基本の開始距離は10です。
*   ダスト: 捨て札置き場にある結晶。様々なコストや効果で移動します。

### 2. カードの種類
主に2種類のカードがあります。

*   通常札: 緑色の枠。デッキに7枚入れます。山札から引いて手札として使用します。
*   切札: 赤色の枠。デッキに3枚入れます。裏向きで「切札置き場」に置かれ、フレアを支払うことでいつでも（条件を満たせば）使用できます。使用後は通常、表向き（使用済み）になります。

### 3. 基本動作
自分のターン中、1「集中力」または手札1枚を消費することで、以下の動作を行えます。

1.  前進: 間合からオーラへ結晶を1つ動かす。（間合が0より大きく、オーラが上限未満の場合）。距離を詰めつつ防御を固めます。※達人の間合（2以下）では行えません。
2.  後退: オーラから間合へ結晶を1つ動かす。（オーラがある場合）。距離を取りますが、防御が薄くなります。
3.  纏い（まとい）: ダストからオーラへ結晶を1つ動かす。防御を回復します。
4.  宿し（やどし）: オーラからフレアへ結晶を1つ動かす。切札のためのゲージを溜めます。

### 4. キーワード能力
特定のメガミが持つ特殊なルールです。

*   連火（れんか） - *[ヒミカ](../megami/index.md)*: このターン中に（このカードを含めて）3枚以上のカードを使用していると効果発動。
*   決死（けっし） - *[ユリナ](../megami/index.md)*: 自分のライフが3以下の時に効果発動・強化。
*   八相（はっそう） - *[サイネ](../megami/index.md)*: 自分のオーラが1以下の時に効果発動・強化。（古いルールでは異なる定義でしたが、現在は「オーラ1以下」が主流）。
*   境地（きょうち） - *[トコヨ](../megami/index.md)*: 自分の集中力が1以上（または2）の時に効果発動。
*   設置（せっち） - *[オボロ](../megami/index.md)*: カードを伏せて「設置」し、後のターンで展開する。

## ターンの流れ

1.  [開始フェイズ](flow.md):
    *   集中力を1得る（上限2）。
    *   [再構成](flow.md): 山札が空の場合、または任意で宣言した場合、捨て札を山札に戻してシャッフルします。この時、ライフに1ダメージを受けます（[再構成](flow.md)ダメージ）。
    *   カードを2枚引く（先攻1ターン目は3枚手札から開始し、ドローなし等、細かいルールあり）。

2.  [メインフェイズ](flow.md):
    *   基本動作を行う。
    *   手札からカードを使用する。
    *   切札を使用する。

3.  [終了フェイズ](flow.md):
    *   手札上限（通常は2枚）になるようにカードを捨てる。
    *   「ターン終了時」の効果を処理する。
