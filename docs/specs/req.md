# 桜降る代に決闘を - 要求仕様書

## ゲーム概要

**桜降る代に決闘を**（ふるよに）は、2人のプレイヤーが互いのライフを0にすることを目指す対戦型カードゲーム。各プレイヤーは「メガミ」と呼ばれるキャラクターを2柱選択し、それぞれのカードを組み合わせてデッキを構築する。

---

## 基本ルール

### 勝利条件
- 相手プレイヤーのライフを0にする
- 何らかの効果によって敗北条件を満たした場合、相手が勝利

### ゲームの準備

#### 桜花結晶の配置
- **間合**: 10個
- **オーラ**: 各プレイヤー3個
- **ライフ**: 各プレイヤー8個

#### デッキ構築
- **メガミ選択**: 各プレイヤーは2柱のメガミを選択
- **通常札**: 7枚（山札に配置）
- **切札**: 3枚（切札エリアに裏向きで配置）

#### 初期状態
- **初期手札**: 山札から3枚ドロー（入れ替え可能）
- **集中力**: 先攻0、後攻1

### ターン構造

#### 開始フェイズ
- 集中力を増やす
- 再構成（山札シャッフル、ライフ1ダメージの可能性）
- カードドロー

#### メインフェイズ
- 手札からカードを使用
- 基本動作の実行

#### 終了フェイズ
- ターン終了処理

### 基本動作

| 動作 | 効果 |
|------|------|
| **前進/後退** | 間合の桜花結晶を移動し、相手との距離を調整 |
| **宿し** | オーラからフレアを得る |
| **離脱** | ダストからフレアを得る |
| **纏い** | 山札からカードをドロー |

### カード種別

#### 通常札
- 攻撃、防御、移動などの基本的な行動
- 山札から引いて使用

#### 切札
- 強力な効果を持つカード
- フレア（集中力などを消費して得られるコスト）が必要
- 未使用時は裏向き、使用後は表向き

## ペア仕様

### ペアの概念
2柱のメガミを組み合わせることで、独自の戦術・シナジーが生まれる。各ペアには固有の戦型名が付けられる。

### 主要ペア一覧

#### 1. 刀薙（ユリナ/サイネ）
- **戦型**: 高打点ミッドレンジ
- **得意間合**: 3〜5
- **勝ち筋**: 大型切札2枚 + 条件達成（決死・八相）での火力跳ね上がり
- **強み**: 条件達成時の異常値打点で相手の受けを破壊
- **弱み**: 機動力不足、遠距離レンジロック・超近距離クリンチに弱い

#### 2. 刀扇（ユリナ/トコヨ）
- **戦型**: バランス型
- **特徴**: 攻防バランスが良く、どんな相手にも対応可能
- **強み**: 構築調整の柔軟性、初心者〜上級者まで握り続けられる
- **弱み**: 「困ったら刀扇」になりやすい

#### 3. 薙扇（サイネ/トコヨ）
- **戦型**: 万能コントロール
- **特徴**: 全ペア屈指の防御力、対応札の多さ
- **強み**: 条件（八相・境地）達成時の攻防一致、相手にケアを強制
- **弱み**: 条件管理の精度が必要、雑なプレイでは遅いデッキに

#### 4. 銃扇（ヒミカ/トコヨ）
- **戦型**: 間合支配型
- **特徴**: 遠距離連火 + 近距離対応の両立
- **強み**: 相手の対策方針を迷わせる
- **弱み**: バランス取り過ぎると決め手が薄い

#### 5. 薙銃（サイネ/ヒミカ）
- **戦型**: 遠距離レンジロック
- **特徴**: 後退しながらの攻撃、連火の打点ブースト
- **勝ち筋**: 1巡目と中盤以降に2回の攻撃の波を作る
- **強み**: 距離と受けの両方にケアを強制
- **弱み**: ミスると一気にライフが溶ける

#### 6. 刀忍（ユリナ/オボロ）
- **戦型**: 近距離ビートダウン
- **得意間合**: 2付近
- **特徴**: 設置札によるコンボ + 大型切札
- **強み**: オーラを剥がしてからのリーサルが強力
- **弱み**: 遠距離レンジロック・間合0クリンチに耐性が低い

#### 7. 忍傘（オボロ/ユキヒ）
- **戦型**: テクニカルコントロール
- **特徴**: 設置 + 傘の開閉による得意間合変化
- **勝ち筋**: クロック的に削りながら終盤リーサル
- **強み**: 特定対面（レンジロック相手）に刺さる、回り始めると止まらない
- **弱み**: ループが止まる条件（大被弾・後退不能）で破綻

#### 8. 薙書（サイネ/シンラ）
- **戦型**: ギミック破壊型
- **特徴**: 相手のデッキギミック破壊、悠長なプラン否定
- **強み**: デッキタイプの射程が広い、対面適応力が高い
- **弱み**: 高い知識が必要

#### 9. 銃鎚（ヒミカ/ハガネ）
- **戦型**: 超重量級ビートダウン
- **特徴**: 遠距離攻撃で引きつけ、超火力切札で決着
- **強み**: 対応不能な超打点、決まった時の爽快感
- **弱み**: コンボパーツを揃えるまでのリソース管理が厳しい

#### 10. 扇毒（トコヨ/チカゲ）
- **戦型**: 持久戦型
- **特徴**: トコヨの防御力 + チカゲの毒
- **強み**: 相手のプランを毒でズラし続ける
- **弱み**: 決定打が遅い、火力で押し切られる可能性

## 技術仕様

### フロントエンド構成
- **HTML**: `index.html`（静的ページ）

- **CSS**: Pure CSS（`src/style.css`）

### デザイン要件
- **桜吹雪エフェクト**: 動的アニメーション
- **グラスモーフィズム**: モダンUI
- **レスポンシブ対応**: モバイル/タブレット/デスクトップ
- **ダークモード**: 目に優しい配色

### パフォーマンス要件
- Vanilla JSで軽量・高速
- Lazy Loadingで画像最適化
- Intersection Observerでスムーズなスクロールアニメーション
