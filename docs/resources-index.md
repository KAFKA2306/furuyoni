## 有志ツール

### ふるよにボードシミュレーター

[シミュレーターを開く](https://furuyoni-simulator.herokuapp.com/){ .md-button }

- PCブラウザ向け。
- オンライン対戦と観戦に対応。
- ルール処理や盤面管理の多くは自動化されていない。
- サイト自身の説明では、新幕シーズン8-2の24柱を中心に扱っています。

現行の再演カードリストと一致するとは限りません。

## 動画を探す

- [BakaFire Party公式YouTubeチャンネル](https://www.youtube.com/@bakafireparty916)

動画は投稿日、対象版、シーズンを確認してください。
