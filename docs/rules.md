## 2. 基本構造の見取り図 <a id="basics"></a>

以下はゲームを理解するための非公式な要約です。再演と新幕で細部が異なる場合は、公式ルールを優先してください。

### 桜花結晶の循環

ボード上の**桜花結晶**は、間合・オーラ・ライフ・フレア・ダストなどの領域を移動します。

### 主な領域

1. **[間合](#range)**: 二人の間の距離。
2. **[オーラ](#aura)**: 攻撃を受ける際に利用する領域。
3. **[ライフ](#life)**: 勝敗に関わる領域。
4. **[フレア](#flare)**: 切札のコストに利用する領域。
5. **[ダスト](#dust)**: 共有の結晶領域。

### カードの区分

- **通常札**: 山札を構成し、手札から使用するカード。
- **切札**: フレアを支払って使用するカード。
- **ダメージ X/Y**: オーラダメージとライフダメージの組み合わせ。

### 基本動作 <a id="basic-actions"></a>

新幕では集中力1または手札1枚を支払って基本動作を行います。

- **[前進](#advance)**: 間合から自分のオーラへ結晶を動かす。
- **[後退](#retreat)**: 自分のオーラから間合へ結晶を動かす。
- **[纏い](#matoy)**: ダストから自分のオーラへ結晶を動かす。
- **[宿し](#yadoshi)**: 自分のオーラから自分のフレアへ結晶を動かす。

## 3. ゲームの流れ <a id="game-flow"></a>

ゲームは概ね、メガミの選択、眼前構築、桜花決闘の順に進みます。具体的な選択数・デッキ枚数・処理手順は対象フォーマットの公式ルールを確認してください。

### 新幕アーカイブでの流れ

1. **三拾一捨** <a id="3-choose-1"></a>: 3柱から2柱を選ぶ。
2. **眼前構築** <span id="construction"></span>: 通常札7枚、切札3枚を構築。
3. **決闘**:
    - 開始フェイズ
    - メインフェイズ
    - 終了フェイズ

## 5. 版をまたいで使える戦術の問い <a id="strategy"></a>

カード数値や環境評価ではなく、次の問いを使うと版が変わっても対戦を整理できます。

1. 相手が次のターンに出せる最大打点は何か。
2. 自分はオーラ・ライフ・フレアのどこへ結晶を渡したいか。
3. 相手の対応札をどこまで考慮するか。
4. 山札再構成までに何を達成するか。
5. 眼前構築で勝ち筋と負け筋をどこまで絞れるか。

具体的なリーサル値、Tier、必須カードは対象版と更新時期に依存します。

## 8. FAQ

### どのルールを優先すべきですか

現行の再演を遊ぶ場合は再演公式ルールを優先します。新幕製品や新幕の過去対戦を調べる場合は、新幕公式アーカイブと該当シーズンの資料を確認します。

### 再演古典戦ではこのページだけ見れば遊べますか

いいえ。再演古典戦は現行の公式フォーマットです。使用可能範囲や変更点は再演公式サイトを確認し、このページは旧カードや旧キーワードを探す補助索引として使ってください。

### 対応の処理順を確認したい

厳密な処理順は対象版の総合ルールを確認してください。非公式要約だけで大会裁定を判断しないでください。
