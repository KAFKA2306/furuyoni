# 桜降る代に決闘を：総合ガイド

このガイドでは、基本ルール（メカニクス）から初心者向けカリキュラム、そして中級者向けの戦略までを一貫して解説します。

---

## 1. 序章：闘いの始まり

一つのコンセプトを学ぶためにデザインされた、厳選されたペアのカリキュラムを通じて、あなたを一人前のミコト（決闘者）へと導きます。
各ステップでは、具体的なデッキレシピと立ち回りのガイドを提供します。これらはあくまで一例ですが、そのペアの強みを理解する良い出発点となるでしょう。

---

## 2. 基本ルールとメカニクス <a id="basics"></a>

### 桜花結晶の循環
このゲームの最大の特徴は、ボード上のあらゆるところにあるピンク色の石**「桜花結晶」**です。
体力も、防御も、エネルギーも、距離も、すべてはこの結晶の数で表されます。結晶は「消える」ことはほとんどなく、板の上を**ぐるぐると循環**します。

#### 5つのエリア（領域）
1.  **[間合](#range)（距離）**: 二人の間の距離。基本の開始距離は10です。
2.  **[オーラ](#aura)（盾）**: あなたを守るバリア。最大5つまで。
3.  **[ライフ](#life)（体力）**: これが0になると敗北します。通常は10です。
4.  **[フレア](#flare)（エネルギー）**: 強力な「切札」を使うためのゲージ。
5.  **[ダスト](#dust)（空き地）**: 誰のものものでもない結晶。

### カードの種類と見方
*   **通常札**: 山札から引いて手札として使用します（7枚）。
*   **切札**: [フレア](#flare)を支払うことで使用できる必殺技（3枚）。
*   **ダメージ (X/Y)**: 左側が[オーラ](#aura)へのダメージ、右側が[ライフ](#life)へのダメージ。受ける側がどちらで受けるか選択します。

### <span id="basic-actions">基本動作</span>
[集中力](#focus)1か手札1枚を消費して行います。
*   **[前進](#advance)**: [間合](#range) ⇒ [オーラ](#aura)
*   **[後退](#retreat)**: [オーラ](#aura) ⇒ [間合](#range)
*   **[纏い](#matoy)**: [ダスト](#dust) ⇒ [オーラ](#aura)
*   **[宿し](#yadoshi)**: [オーラ](#aura) ⇒ [フレア](#flare)

---

## 3. 決闘の流れ <a id="game-flow"></a>

1.  **三拾一捨** <a id="3-choose-1"></a>: 3柱から2柱を選ぶ。
2.  **眼前構築** <span id="construction"></span>: 通常札7枚、切札3枚を構築。
3.  **決闘**:
    *   **開始フェイズ**: [集中力](#focus)+1、ドロー2枚、[再構成](#reconstruction)（任意/強制）。
    *   **メインフェイズ**: カード使用、基本動作。
    *   **終了フェイズ**: 手札調整。

---

## 4. 初心者修練路 (ステップ 1-8) <a id="curriculum"></a><a id="shousou"></a>

ここでは、代表的なメガミのペアを通じてゲームの深淵を学びます。

> [!TIP]
> **カード詳細について**: 各ステップで使用するメガミの全カードは [カード一覧](megami/cards.md) で確認できます。ここでは重要なピックアップカードのみを記載します。

### ステップ 1: 目覚め (刀薙 - [ユリナ](megami/01_yurina.md)/[サイネ](megami/02_saine.md))
*   **テーマ**: 基礎と攻撃。
*   **コンセプト**: [前進](#advance)し、攻撃し、リソース管理を学ぶ。
*   **重要レッスン**: 「[間合](#range)の管理」。[ユリナ](megami/01_yurina.md)は3-4、[サイネ](megami/02_saine.md)は4-5が得意です。 {: #yurina-saine }

### ステップ 2: 平衡 (刀扇 - [ユリナ](megami/01_yurina.md)/[トコヨ](megami/04_tokoyo.md))
*   **テーマ**: 攻防のバランス。
*   **コンセプト**: 防御とカウンターを学ぶ。
*   **重要レッスン**: 「リソース効率」。[トコヨ](megami/04_tokoyo.md)の「久遠ノ花」のために[フレア](#flare)を残す動き。

### ステップ 3: 制御 (薙扇 - [サイネ](megami/02_saine.md)/[トコヨ](megami/04_tokoyo.md))
*   **テーマ**: 忍耐とカウンター。
*   **コンセプト**: 相手のミスを待ち、鉄壁の[オーラ](#aura)でリソースを枯渇させる。

## 5. 勝利への戦略指南 <a id="strategy"></a>

### 相手の「最大値」を見積もる
デッキ構築（[眼前構築](#construction)）では、相手ができる「一番強い動き」を想像しましょう。
*   **リーサルライン**: [ユリナ](megami/01_yurina.md)なら「月影落」で6点前後など、危険域を把握します。
*   **防御性能**: [トコヨ](megami/04_tokoyo.md)がいるなら、大技は通らない前提で動く必要があります。

### 資源の循環：桜花結晶の法則
*   **ダメージ選択のジレンマ**: 「あえて[ライフ](#life)で受けて、返しのターンに切札で仕留める」判断が中級者への第一歩です。
*   **[集中力](#focus)と[ライフ](#life)の天秤**: [ライフ](#life)が4〜5になったら危険信号です。
