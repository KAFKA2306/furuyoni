## 版をまたいで確認する手順

1. この索引で旧版のメガミ・カード名を特定する。
2. [新幕からの公式変更点](https://furuyoni.sekiseiro.com/re/rules/renewal/)で変更対象か確認する。
3. [再演カードリスト](https://furuyoni.sekiseiro.com/re/cardlist/)で現行テキストを確認する。
4. 遊ぶフォーマットの使用可能範囲・禁止改定を[再演公式ルール](https://furuyoni.sekiseiro.com/re/rules/)で確認する。
