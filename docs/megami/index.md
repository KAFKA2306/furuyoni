## メガミ別詳細ガイド

### <a id="yurina"></a>01 ユリナ
- **難易度**: ★☆☆☆☆
- **得意間合**: 3-4
- **特徴**: 初心者に最も推奨されるメガミ。シンプルに「斬って勝つ」スタイル。ライフが減ると「決死」状態になり、全ての攻撃カードが凶悪な性能へ変貌します。
- **代表札**: 『斬』『一閃』『月影落』
- **攻略のヒント**: 間合3-4を維持し、常に相手のオーラを削り続けましょう。決死圏内まで耐えてからの『月影落』は最強の逆転手段です。

### <a id="saine"></a>02 サイネ
- **難易度**: ★★☆☆☆
- **得意間合**: 4-5
- **特徴**: 対応（カウンター）の名手。八相（オーラ1以下）で真価を発揮するため、あえて攻撃を受ける勇気も必要です。攻防のバランスが非常に良いです。
- **代表札**: 『八方振り』『圏域』『氷雨細音の果ての果て』
- **攻略のヒント**: 相手の攻撃に合わせて『圏域』や『見切り』で翻弄し、オーラを減らして「八相」バフを受けた強力な一撃を狙いましょう。

### <a id="himika"></a>03 ヒミカ
- **難易度**: ★★☆☆☆
- **得意間合**: 5-8
- **特徴**: 防御を捨てた超攻撃型。連火能力により、1ターンに大量のカードを叩き込む「バーストターン」を作ることが勝利の鍵です。
- **代表札**: 『シュート』『マグナムカノン』『レッドバレット』
- **攻略のヒント**: 可能な限り後ろに下がり、手札を溜め込んで一気に解放します。防御はほぼ期待できないため、「殺られる前に殺す」精神が重要です。

### <a id="tokoyo"></a>04 トコヨ
- **難易度**: ★★★☆☆
- **得意間合**: 2-4
- **特徴**: コントロール型の頂点。集中力2の状態「境地」を維持することで、鉄壁の防御を誇ります。
- **代表札**: 『雅打ち』『梳流し』『久遠ノ花』
- **攻略のヒント**: 集中力を切らさず、相手の隙を見て『梳流し』でじわじわ削りましょう。再構成ダメージを恐れず、長期戦に持ち込めば勝利が見えます。

### <a id="oboro"></a>05 オボロ
- **難易度**: ★★★★☆
- **得意間合**: 2-3
- **特徴**: 「設置」により、本来使えないタイミング（攻撃直後や相手ターン中）でカードを発動させます。
- **代表札**: 『鋼糸』『影菱』『熊介』
- **攻略のヒント**: 伏せ札を「リソース」として活用します。切札『熊介』によるバーストダメージは非常に強力ですが、事前の準備が重要です。

## 2柱の組み合わせ戦略（シナジー）

『ふるよに』の醍醐味は、2柱を組み合わせる「双掌」にあります。代表的なシナジーパターンを紹介します。

### 1. 役割分担型（アタッカー ＋ サポーター）
片方が火力を出し、もう片方が移動や防御を補う、最も安定した構成。
- **ユリナ ＋ サリヤ**: サリヤの「騎動」で間合を一気に詰め、ユリナの重い一撃を叩き込む。
- **ヒミカ ＋ トコヨ**: ヒミカが遠距離から撃ち、トコヨが近寄られた際の防御を固める。

### 2. コンボ重視型（相乗効果）
特定のメカニズムを共有したり、相互にリソースを供給する構成。
- **ライラ ＋ オボロ**: オボロの手数の多さでライラのゲージを高速で溜め、後半に圧倒する。
- **チカゲ ＋ コルヌ**: 毒で手札を封じ、凍結でオーラを封じる「完全ロック」狙い。

### 3. レンジロック型（間合の共有）
得意な間合が一致しているペア。その間合いを維持し続けることで優位を保つ。
- **サイネ ＋ ヒミカ**: 間合5周辺での戦いを強要し、中遠距離から一方的に削る。
- **ユキヒ ＋ オボロ**: 間合2での接近戦に特化し、相手を懐から逃がさない。
