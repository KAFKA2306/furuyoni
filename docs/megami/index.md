# メガミ一覧

現在使用可能な主要24柱のメガミ一覧です。それぞれ固有のキーワード能力と独自の戦術を持っています。

## メガミ・マスター・ロスター

| No. | 名前 | 権能 | 戦術の核心 |
|:---:|:---|:---|:---|
| 01 | [ユリナ](01_yurina.md) | 刀 | 圧倒的な攻撃力。間合3-4を死守し、高打点の攻撃を連打する。 |
| 02 | [サイネ](02_saine.md) | 薙刀 | 堅実なリーチと対応。間合4-5で安定して戦い、対応札で受け流す。 |
| 03 | [ヒミカ](03_himika.md) | 銃 | 超遠距離からの弾幕。間合5-8で一方的に攻撃する。 |
| 04 | [トコヨ](04_tokoyo.md) | 扇 | 鉄壁の防御と制御。集中力を高めて相手の攻撃を無効化・反射する。 |
| 05 | [オボロ](05_oboro.md) | 忍 | 技巧的な連続攻撃。伏せ札からの不意打ちや再利用で手数負けしない。 |
| 06 | [ユキヒ](06_yukihi.md) | 傘 | 中近距離のレンジ制御。傘を閉じれば近距離、開けば中距離を支配する。 |
| 07 | [シンラ](07_shinra.md) | 書 | 盤面と情報の支配。相手の行動を制限し、強引に自分の土俵に引き込む。 |
| 08 | [ハガネ](08_hagane.md) | 槌 | 一撃必殺の重機動。大移動から繰り出す高火力攻撃でオーラごと砕く。 |
| 09 | [チカゲ](09_chikage.md) | 毒 | 持続的なリソース妨害。毒カードで相手の手札や精神を削り取る。 |
| 10 | [クルル](10_kururu.md) | 機巧 | 爆発的なコンボ構築。特殊な発動条件を満たし、即死級の恩恵を得る。 |
| 11 | [サリヤ](11_sariya.md) | 乗騎 | 高機動・高出力の代償。造花結晶を燃料として消費し加速する。 |
| 12 | [ライラ](12_laila.md) | 爪 | 風雷を纏う連撃。カードを使うほどゲージが溜まり、攻撃が激化する。 |
| 13 | [ウツロ](13_utsuro.md) | 鎌 | 死滅から這い出る魔王。ダストが増える終盤、全能力が覚醒する。 |
| 14 | [ホノカ](14_honoka.md) | 旗 | 精霊との共鳴・成長。カードを「開花」させ、後半に向けて強化する。 |
| 15 | [コルヌ](15_korunu.md) | 橇 | 凍てつくリソースロック。相手のオーラやフレアを凍結して奪う。 |
| 16 | [ヤツハ](16_yatsuha.md) | 鏡 | 変幻自在のコピー術。相手のステータスや攻撃を自分に写し取る。 |
| 17 | [ハツミ](17_hatsumi.md) | — | 情報不足のため詳細調査中。 |
| 18 | [ミズキ](18_mizuki.md) | 兜 | 対応特化。大手楯無門と防壁で堅牢な守りを築く。 |
| 19 | [メグミ](19_megumi.md) | 唐棹 | 高火力アタッカー。瀧川希の掌で攻撃を強化し圧殺する。 |
| 20 | [カナヱ](20_kanawe.md) | — | 情報不足のため詳細調査中。 |
| 21 | [カムヰ](21_kamui.md) | 剣 | 通常札の圧倒的攻撃力でテンポを取る。禁忌ゲージの管理が鍵。 |
| 22 | [レンリ](22_renri.md) | — | 情報不足のため詳細調査中。 |
| 23 | [アキナ](23_akina.md) | 算盤 | リソース操作の王。回収と投資で経済的に支配する。 |
| 24 | [シスイ](24_shisui.md) | 鋸 | 裂傷による近距離ビートダウン。オーラ上限を削り、圧倒的打点を叩き込む。 |
