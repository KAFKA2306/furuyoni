# メガミ一覧（第6〜7シーズン期）

現在使用可能な主要16柱のメガミ一覧です。それぞれ固有のキーワード能力とプレイスタイルを持っています。

| No. | 名前 | 権能 | キーワード | 戦術・特徴 |
|:---:|:---|:---|:---|:---|
| **01** | **ユリナ**<br><img src="../assets/images/chara/ico-01.png" width="50"> | 刀 | **決死**（ライフ3以下） | **ビートダウン / 高打点**。シンプルかつ強力。間合3-4に潜り込み、重い一撃（月影落）を叩き込む。放置すれば殴られ、反撃すれば決死でさらに強くなる。 |
| **02** | **サイネ**<br><img src="../assets/images/chara/ico-02.png" width="50"> | 薙刀 | **八相**（オーラ1以下） | **中距離 / カウンター**。間合4-5での安定した攻撃に加え、相手の攻撃に対する強力な対応札（氷雨細音の果ての果て）を持つ。どのメガミとも組みやすい万能型。 |
| **03** | **ヒミカ**<br><img src="../assets/images/chara/ico-03.png" width="50"> | 銃 | **連火**（3枚以上使用） | **コンボ / 遠距離**。間合4-8から一方的に攻撃し、相手のライフを削る（マグナムカノン）。手札を溜め込んでからのバーストダメージが得意だが、防御は脆い。 |
| **04** | **トコヨ**<br><img src="../assets/images/chara/ico-04.png" width="50"> | 扇 | **境地**（集中力2以上） | **コントロール / 守り**。鉄壁の防御を誇る。攻撃を無効化したり反射したり（久遠ノ花）する。相手のデッキ切れを狙ったり、確実なカウンターで勝利する。 |
| **05** | **オボロ**<br><img src="../assets/images/chara/ico-05.png" width="50"> | 忍 | **設置**（伏せて準備） | **アグロ / 奇襲**。攻撃後に手札に戻るカードや、伏せカード（設置）を利用して相手を翻弄する。間合2での手数が多い。対応不可の攻撃（壬蔓）なども持つ。 |
| **06** | **ユキヒ**<br><img src="../assets/images/chara/ico-06.png" width="50"> | 傘 | **開閉**（傘の状態変化） | **柔軟 / 変幻自在**。傘が開いているか閉じているかで攻撃範囲が変わる。中距離で牽制しつつ、隙を見て懐に潜り込む（ゆらりび）。操作難易度は高め。 |

| **16** | **ヤツハ**<br><img src="../assets/images/chara/ico-16.png" width="50"> | 鏡 | **鏡映**（同数参照など） | **コピー / カウンター**。相手の攻撃をコピーしたり、攻撃された時に反射したりする（四葉鏡のわらべ唄）。「鏡合わせ」のような戦いを得意とする。 |

## ペアの選び方
基本の決闘では**2柱のメガミ**を選びます。
互いの弱点を補うような組み合わせが推奨されます。

*   **ユリナ（攻撃）** + **サリヤ（機動）** = 「足の速い重戦車」。
*   **トコヨ（防御）** + **チカゲ（妨害）** = 「毒を盛って守り切る」。
*   **ヒミカ（遠距離）** + **サイネ（中距離）** = 「赤と青の支配」。
